//　　以下之シ`ンはム`ビ`で之想定で。
//　　克劳德机、奥蒂莎机、布库尔机を兢à胩乩锼固购拧
//　　巨大龙卷が次第に近付いて来て、いよいよ作战がスタ`ト。
//　　特里斯坦号之道を切り_くため
//　　他之３机がエンジンを震わせて先行してゆく。
//　　\っていた拉莱拉、菲奥娜、皮皮洛が魔法をk婴丹弧
//　　それぞれ之魔力が膨れ上がった上で、巨大な魔法と化す。
//　　３つ之魔法、回方向とは反之ベクトルから、
//　　龙卷之一点に向けてそれぞれぶち当たり、`びを生み出す。
//　　それに目欷堡啤⒁荬送护棉zんでいく特里斯坦号。
//　　しかし`びはすぐにり、翻弄されそうになる特里斯坦。
//　　ここで艾雯之光・・地・水・火・风之魔法がk印
//　　七色之Y界となって特里斯坦号を包みzむ。
//　　そ之まま一荬肆卷之中を突きiけてゆく特里斯坦号。
//　　そして龙卷をiけた所で、螺旋要塞梅尔塞迪克がFれる。
//　　特里斯坦号、龙卷之内壁と螺旋要塞之间を旋回しながら、
//　　そ之ままゆっくりと画面フェ`ドアウト。
//──────────────────────────────
